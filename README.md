# Basic-Python-Language-For-Data-Science
Repository ini berisi materi dasar-dasar Python yang digunakan untuk Data Science.


Tipe Data Dasar: Integer, Float, String, Boolean, None

Operator: Penjumlahan, Pengurangan, Perkalian, Pembagian, Modulus, Floor Division, Exponentiation

String & List: Manipulasi string, indexing, slicing, method dasar, list operations (append, extend, pop, remove, reverse)

Dictionary: Struktur data key-value, method keys(), values(), items(), get(), update()

Kontrol Alur: Conditional statements, loops (for, while), break, continue, pass

Function: User-defined functions, positional, keyword, default, variable length arguments (*args, *kwargs)

NumPy Basics: One-dimensional array, two-dimensional array, row & column vector, transpose, shape, size, ndim, zeros, full, randint

Matplotlib Basics: Line plot, marker, color, linewidth, title, xlabel, ylabel, font customization

Pandas Basics: Series, DataFrame, head(), tail(), indexing, slicing, basic operations

Repository ini cocok untuk pemula yang ingin memahami Python secara mendasar sebelum masuk ke analisis data, visualisasi, dan machine learning.
